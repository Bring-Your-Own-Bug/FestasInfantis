# Controle de Festas Infantis

Esta aplicação permite o usuário controlar aluguéis de festas infantis, podendo cadastrar clientes, temas, itens para cada tema, festas e aluguéis.

## Funcionalidades

- Inserir, editar e excluir clientes
- Inserir, editar e excluir temas
- Inserir, editar e excluir festas
- Inserir, editar e excluir aluguéis
- Inserir itens no tema, podendo adicionar preço diretamente no item que facilitará o calculo final do aluguel
- Cálculo automatico do aluguel conforme o preço total do tema e o preço da mão de obra
- Tabelas zebradas para melhor visualização dos cadastros
